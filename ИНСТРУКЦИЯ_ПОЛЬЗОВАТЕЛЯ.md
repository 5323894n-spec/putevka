# Инструкция пользователя

Программа предназначена для ведения базы водителей и автобусов, составления месячного графика работы, формирования ежедневного наряда, выписки путевых листов, печати и сохранения журналов.

Адрес программы: http://127.0.0.1:5180/

API программы: http://127.0.0.1:8010/

## 1. Общий порядок работы

Рекомендуемый ежедневный порядок:

1. Заполнить справочники во вкладке **База данных**.
2. Составить график во вкладке **График месяца**.
3. Сформировать наряд за выбранную дату.
4. Выписать путевые листы из наряда.
5. Распечатать путевые листы.
6. После возвращения автобуса закрыть путевой лист.
7. Сохранить или распечатать журнал.

## 2. Выбор рабочей даты

В верхней панели программы находится поле даты.

Эта дата используется для:

- просмотра наряда;
- просмотра путевых листов;
- формирования наряда из графика;
- проверки предупреждений по документам и транспорту.

Перед формированием наряда или путевых листов убедитесь, что выбрана нужная дата.

## 3. Заполнение базы данных

Откройте вкладку **База данных**.

Внутри есть разделы:

- **Водители**;
- **Транспорт и ГРЗ**;
- **Маршруты**;
- **Выпуски**.

### 3.1. Добавление водителя

Откройте **База данных** -> **Водители** -> **Добавить водителя**.

Заполните основные поля:

- ФИО;
- табельный номер;
- телефон;
- категория водительского удостоверения;
- номер водительского удостоверения;
- срок действия водительского удостоверения;
- срок действия медсправки;
- колонна;
- статус.

Нажмите **Сохранить**.

### 3.2. Добавление автобуса

Откройте **База данных** -> **Транспорт и ГРЗ** -> **Добавить автобус**.

Заполните:

- государственный регистрационный знак;
- гаражный номер;
- марку;
- модель;
- срок действия диагностической карты;
- статус;
- норму расхода топлива;
- остаток топлива;
- текущий пробег.

Нажмите **Сохранить**.

### 3.3. Добавление маршрута

Откройте **База данных** -> **Маршруты** -> **Добавить маршрут**.

Заполните:

- номер маршрута;
- наименование маршрута;
- вид сообщения;
- протяженность;
- количество рейсов;
- плановый пробег;
- время начала и окончания работы;
- контрольные точки.

Нажмите **Сохранить**.

### 3.4. Добавление выпуска

Откройте **База данных** -> **Выпуски** -> **Добавить выпуск**.

Заполните:

- маршрут;
- номер выпуска;
- время выезда из парка;
- время начала работы;
- время окончания работы;
- время заезда в парк;
- плановый пробег;
- плановое количество рейсов;
- требуемый тип автобуса.

Нажмите **Сохранить**.

## 4. Составление графика на месяц

Откройте вкладку **График месяца**.

Нажмите **Добавить назначение**.

Заполните:

- дату работы;
- колонну;
- маршрут;
- выпуск;
- водителя;
- автобус / ГРЗ;
- смену;
- время выезда;
- время заезда;
- количество рейсов;
- плановый пробег;
- примечание при необходимости.

Нажмите **Сохранить**.

Назначение появится в таблице графика месяца.

## 5. Формирование наряда из графика

1. В верхней панели выберите дату, на которую нужно сформировать наряд.
2. Откройте вкладку **График месяца**.
3. Нажмите **Сформировать наряд за дату**.
4. Перейдите во вкладку **Наряд**.

Все назначения из графика за выбранную дату будут перенесены в наряд.

Если наряд уже был сформирован, программа не создает повторные строки для того же водителя, автобуса, выпуска и смены.

## 6. Ручное добавление строки наряда

Если нужно добавить автобус или водителя без графика:

1. Откройте вкладку **Наряд**.
2. Нажмите **Добавить строку**.
3. Выберите маршрут, выпуск, водителя и автобус.
4. Укажите смену, время выезда, время заезда, пробег и рейсы.
5. Нажмите **Сохранить**.

## 7. Выписка путевых листов

1. Откройте вкладку **Наряд**.
2. Проверьте строки наряда.
3. Нажмите **Выписать путевые листы**.
4. Перейдите во вкладку **Путевые листы**.

Программа автоматически присвоит путевым листам сквозные номера, например:

- `ПЛ-2026-000001`;
- `ПЛ-2026-000002`.

## 8. Печать путевого листа

Откройте вкладку **Путевые листы**.

В строке нужного путевого листа нажмите кнопку с иконкой принтера.

Откроется печатная форма. Нажмите **Печать**.

## 9. Закрытие путевого листа

После возвращения автобуса:

1. Откройте вкладку **Путевые листы**.
2. Нажмите **Закрыть** напротив нужного путевого листа.
3. Заполните:
   - одометр при заезде;
   - выдано топлива;
   - остаток топлива при заезде.
4. Нажмите **Закрыть**.

Программа рассчитает:

- фактический пробег;
- расход топлива по норме;
- фактический расход топлива;
- остаток топлива автобуса.

Статус путевого листа станет **закрыт**.

## 10. Учет топлива

Откройте вкладку **Топливо**.

В таблице отображаются:

- номер путевого листа;
- автобус;
- остаток топлива при выезде;
- выдано топлива;
- остаток топлива при заезде;
- расход по норме;
- фактический расход;
- экономия или перерасход.

Формула фактического расхода:

```text
Фактический расход = Остаток при выезде + Выдано топлива - Остаток при заезде
```

## 11. Сохранение в Excel

В верхней панели есть кнопки:

- **Наряд** - скачать наряд в Excel;
- **Журнал** - скачать журнал путевых листов в Excel.

Файлы сохраняются через браузер в папку загрузок.

## 12. Панель

Вкладка **Панель** показывает:

- количество автобусов на линии;
- количество водителей в наряде;
- количество выписанных путевых листов;
- количество закрытых путевых листов;
- количество незакрытых путевых листов;
- предупреждения по документам и транспорту.

## 13. Предупреждения

Программа предупреждает, если:

- у водителя истек срок водительского удостоверения;
- у водителя истекла медсправка;
- у автобуса истекла диагностическая карта;
- автобус находится в ремонте;
- остаток топлива отрицательный;
- водитель назначен несколько раз на одну дату;
- автобус назначен несколько раз на одну дату.

## 14. Типовой рабочий сценарий

Для начала месяца:

1. Заполните **База данных**.
2. Внесите назначения во вкладке **График месяца** на нужные даты.

Для каждого рабочего дня:

1. Выберите дату в верхней панели.
2. Откройте **График месяца**.
3. Нажмите **Сформировать наряд за дату**.
4. Откройте **Наряд**.
5. Нажмите **Выписать путевые листы**.
6. Откройте **Путевые листы** и распечатайте их.
7. После возвращения автобусов закройте путевые листы.
8. При необходимости скачайте **Журнал** в Excel.

## 15. Если данные не появились

Если после сохранения запись не видна:

1. Нажмите кнопку обновления в верхней панели.
2. Проверьте, что открыта правильная дата.
3. Обновите страницу браузера.

Рабочий адрес программы: http://127.0.0.1:5180/
