# Шаблоны

Сюда можно положить утвержденные организацией шаблоны Excel/PDF:

- `путевой лист.xls`
- `наряд.xls`
- PDF-образец для визуального контроля печатной формы

В MVP экспорт формируется программно через `openpyxl`. Следующий шаг - привязать заполнение к конкретным ячейкам утвержденного шаблона.
